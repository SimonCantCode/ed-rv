# ed-rv
[SEVERELY WIP lol] (Intended to become) an [ed](https://en.wikipedia.org/wiki/Ed_(text_editor)) inspired text editor written in RISC-V assembly.

## How to run
Compile with ```make```, run with ```qemu-riscv64 ed-rv```.

## Commands
* ```q``` to quit
* ```@``` to print (8 bytes) of file contents
